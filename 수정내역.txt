버전: 1.2 (2025-10-19 14:48)

【 수정 사항 2: 공백 처리 개선 】

문제:
  제거/추가 기능에서 사용자가 입력한 공백이 무시되는 문제
  예: " [공금]" 제거 시 공백이 남아 "01권 .zip" 형태가 됨

수정:
  사용자가 입력한 공백을 포함하여 정확히 제거/추가되도록 수정

  제거 예시:
    - "[공금]" 입력 → "01권 [공금].zip" → "01권 .zip" (공백 남음)
    - " [공금]" 입력 → "01권 [공금].zip" → "01권.zip" (공백 제거됨)

  추가 예시:
    - "[공금]" 입력 → "[공금]사카모토.zip" (공백 없음)
    - "[공금] " 입력 → "[공금] 사카모토.zip" (공백 있음)

수정 파일:
  - file_renamer.py (remove_text, add_text 메서드 수정)

【 테스트 결과 】

✓ 공백 포함 제거 테스트 통과
✓ 공백 포함 추가 테스트 통과
✓ 확장자 앞 공백이 올바르게 제거됨
✓ 연속 공백이 하나로 정리됨
