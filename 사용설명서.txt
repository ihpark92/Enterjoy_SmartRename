       엔쪼 스마트리네임 - 사용 설명서
       (Smart File Renamer with Pattern Recognition)
═══════════════════════════════════════════════════

【 프로그램 위치 】
실행 파일: dist\Enterjoy_SmartRename.exe

【 테스트 파일 】
테스트용 샘플 파일: test_files\ 폴더

【 사용 방법 】

1. 프로그램 실행
   - dist\Enterjoy_SmartRename.exe 더블 클릭

2. 폴더 선택
   - "폴더 선택" 버튼 클릭
   - 파일명을 변경할 파일들이 있는 폴더 선택
   - 예: test_files 폴더 선택

3. 패턴 확인 및 선택
   - 자동으로 검출된 패턴들이 라디오 버튼으로 표시됨
   - 원하는 패턴을 선택
   - 예: "사카모토 데이즈 01권.zip" 선택

4. 미리보기 확인
   - 테이블에서 원본 파일명과 변경될 파일명 확인
   - 변경되는 파일은 노란색으로 표시됨

5. 추가 편집 (선택사항)

   5-1. [제거] 기능
        - 입력창에 제거할 텍스트 입력
        - 예: " [공금]" 입력 (앞에 공백 포함)
        - "적용" 버튼 클릭
        - 모든 파일명에서 해당 텍스트가 공백 포함하여 제거됨

        ⚠️ 중요: 공백도 입력한 그대로 제거됩니다!
        - "[공금]" 입력 → "01권 [공금].zip" → "01권 .zip" (공백 남음)
        - " [공금]" 입력 → "01권 [공금].zip" → "01권.zip" (공백 없음)

   5-2. [추가] 기능
        - 입력창에 추가할 텍스트 입력
        - "앞" 또는 "뒤" 라디오 버튼 선택
        - "적용" 버튼 클릭
        - 모든 파일명에 텍스트가 추가됨

        ⚠️ 중요: 공백도 입력한 그대로 추가됩니다!
        - "[공금]" 입력 (앞) → "사카모토.zip" → "[공금]사카모토.zip" (공백 없음)
        - "[공금] " 입력 (앞) → "사카모토.zip" → "[공금] 사카모토.zip" (공백 있음)

6. 파일명 변경 실행
   - "파일명 변경 실행" 버튼 클릭
   - 확인 메시지에서 "Yes" 클릭
   - 변경 완료 메시지 확인

【 예제 시나리오 】

시나리오 1: 모든 파일을 깔끔한 패턴으로 통일
─────────────────────────────────────────────
원본:
  사카모토 데이즈 01권.zip
  [공금] 사카모토 데이즈 06권.zip
  사카모토 데이즈 09권 [절공].zip
  사카모토 데이즈 11.zip

→ "사카모토 데이즈 01권.zip" 패턴 선택

결과:
  사카모토 데이즈 01권.zip
  사카모토 데이즈 06권.zip  (앞의 [공금] 제거됨)
  사카모토 데이즈 09권.zip  (뒤의 [절공] 제거됨)
  사카모토 데이즈 11권.zip  (권 추가됨)


시나리오 2: 모든 파일에 [공금] 태그 추가
─────────────────────────────────────────────
1. "사카모토 데이즈 01권.zip" 패턴 선택으로 통일
2. [추가] 입력창에 "[공금]" 입력
3. "앞" 선택
4. "적용" 클릭

결과:
  [공금] 사카모토 데이즈 01권.zip
  [공금] 사카모토 데이즈 02권.zip
  ...


【 주의사항 】

⚠ 파일명 변경은 되돌릴 수 없습니다!
⚠ 중요한 파일은 백업 후 사용하세요
⚠ 미리보기를 충분히 확인한 후 실행하세요
⚠ 파일명에는 다음 문자를 사용할 수 없습니다: < > : " / \ | ? *


【 문제 해결 】

Q: 패턴이 검출되지 않아요
A: 파일명에 숫자가 포함되어 있는지 확인하세요

Q: 파일명 변경이 실패해요
A: 파일이 다른 프로그램에서 열려있지 않은지 확인하세요

Q: 파일명 충돌 오류가 발생해요
A: 변경 후 같은 이름의 파일이 생기는 경우입니다.
   패턴을 다시 확인하거나 일부 파일을 다른 폴더로 이동하세요
